# inko

not me over-engineering my clock in/out process

## Usage


### Run

You should get prompted to input "in" or "out" by running:

    ruby inko.rb

## WSL caveats:

You need to install https://sourceforge.net/projects/vcxsrv/ and run everytim before running Inko to open google chrome from WSL
